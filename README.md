# 🎓 Student App - Academic Companion

A comprehensive, production-quality Student App built with **vanilla HTML5, CSS3, and JavaScript**. Mobile-first, PWA-ready, and Firebase-compatible.

---

## ✨ Features

### 📚 Academic Management
- **Dashboard** - Overview of subjects, assignments, schedule
- **Subjects** - Track all subjects with attendance & progress
- **Timetable** - Weekly class schedule with time slots
- **Calendar** - Monthly view with events & deadlines
- **Attendance** - Per-subject attendance tracking with charts
- **Assignments** - Track submissions & deadlines
- **Homework** - Daily homework management
- **Notes** - Create & organize study notes
- **Results** - Semester results & grade tracking
- **Progress** - Learning streak & achievements

### 🎯 Practice & Testing
- **Quizzes** - Interactive quizzes with timer
- **Mock Tests** - Full-length practice exams
- **AI Assistant** - Chat-based study helper

### 👥 Community
- **Teachers** - Faculty directory with contact info
- **Notifications** - Stay updated with alerts
- **Feedback** - Rate & suggest improvements

### 🛠️ Technical Features
- 📱 **Mobile-First** responsive design
- 🌗 **Dark/Light/Auto** theme support
- 📲 **PWA** - Install on home screen
- 📡 **Offline Support** via Service Worker
- 🔥 **Firebase-Ready** architecture
- 💾 **Local Storage** for data persistence
- ♿ **Accessible** - ARIA labels, keyboard nav
- ⚡ **Fast** - No frameworks, pure vanilla JS
